🛡️ PhishGuard AI: Next-Gen Phishing Protection
Developed for AI4Dev '26 @ PSG College of Technology, Coimbatore

PhishGuard AI is a proactive cybersecurity suite that moves beyond reactive blacklists. By utilizing a BERT-based Transformer model fine-tuned for phishing heuristics, our system identifies malicious intent in real-time. To ensure a seamless user experience, we have optimized the inference engine using NVIDIA CUDA, allowing the model to run directly on the local GPU.


Zero-Cloud Privacy: 100% local inference ensures no browsing data ever leaves the machine.

Hardware Optimized: High-performance backend using FastAPI and Uvicorn, optimized for the GTX 1650 baseline.

Seamless Deployment: Custom Windows installer (v1.5) with automated browser extension mapping to C:\ for rapid setup.
